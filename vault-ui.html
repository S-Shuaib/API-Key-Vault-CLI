API KEY
VAULT

An encrypted local CLI vault for storing, retrieving, and managing API keys by project — with AES-256-GCM encryption, PBKDF2 key derivation, and zero cloud dependency.

Python 3.8+ cryptography SQLite AES-256-GCM PBKDF2-HMAC-SHA256
🔐
AES-256
GCM · ENCRYPTED
Interactive Demo
bash — api-key-vault
Security Model
🔒 Encryption AES-256-GCM (authenticated)
🧂 Key Derivation PBKDF2-HMAC-SHA256
🎲 Salt 32 bytes, random per vault
🎯 Nonce 12 bytes, random per value
🔁 Iterations 600,000 (NIST 2023 min)
💾 Storage ~/.api_vault/ (chmod 600)
🚫 Master password Never stored. Canary only.
Command Reference
vault.py initSet up vault
vault.py add <proj> <key>Store a secret
vault.py get <proj> <key>Retrieve a secret
vault.py listAll projects
vault.py list <proj>Keys in project
vault.py delete <proj> <key>Remove a key
vault.py nuke <proj>Delete project
vault.py export <proj>Write .env file
vault.py change-passwordRotate master key
